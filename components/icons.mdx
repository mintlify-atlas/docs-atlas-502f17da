Icons are inline components that display icons from the Lucide icon library with customizable sizes.

## Usage

Insert an icon using the `/` menu:

```
/icon
```

Or use the component tag:

```
<icon name="FileText" size="20" />
```

## Features

- **Lucide icons** - Access to the entire Lucide icon library
- **Customizable size** - Set icon size in pixels
- **Dynamic loading** - Icons load on-demand from Lucide
- **Automatic normalization** - Icon names auto-convert to PascalCase
- **Inline display** - Use icons within text

## Attributes

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `name` | string | `"FileText"` | Lucide icon name (e.g., `"Star"`, `"Heart"`, `"Book"`) |
| `size` | number | `20` | Icon size in pixels |

## Common Icon Sizes

- **16px** - Small icons for tight spaces
- **20px** - Default, good for inline use
- **24px** - Medium icons for emphasis
- **32px** - Large icons for headers
- **48px+** - Extra large for hero sections

## Editing Icons

In edit mode:

1. Click the icon to open the editor
2. Enter a new icon name
3. Adjust the size in pixels
4. Icons update in real-time

## Icon Name Formats

Icons support multiple name formats:

- **PascalCase**: `FileText`, `ArrowRight`
- **kebab-case**: `file-text`, `arrow-right`
- **snake_case**: `file_text`, `arrow_right`
- **camelCase**: `fileText`, `arrowRight`

All formats are automatically normalized to PascalCase.

### Dynamic Icon Loading

Icons use the same dynamic loading system as cards:

From `components/card.tsx`:

```typescript
// card.tsx:183-216
export function resolveInlineIcon(
  icon: ReactNode | CardIconName | undefined,
  options: {
    size?: string | number;
    showIcon?: boolean;
    fallbackIcon?: string;
  } = {},
): ReactNode {
  const { size = 20, showIcon = true, fallbackIcon = DEFAULT_CARD_ICON } = options;
  
  if (!showIcon) return null;
  
  const sizeClass = typeof size === "number" ? `w-[${size}px] h-[${size}px]` : size;
  const sizeStyle = typeof size === "number" ? { width: `${size}px`, height: `${size}px` } : {};
  
  if (typeof icon === "string") {
    return <DynamicIcon iconName={icon} size={sizeClass} fallbackIcon={fallbackIcon} />;
  }
  
  if (icon) {
    return (
      <span className={`${sizeClass} text-muted-foreground`} style={sizeStyle}>
        {icon}
      </span>
    );
  }
  
  const pixelSize = typeof size === "number" ? size : 20;
  return <FileText className="text-muted-foreground" width={pixelSize} height={pixelSize} />;
}
```

### Input Rule

```typescript
// icon.tsx:129-154
export const addIconRule = new InputRule(
  /^<icon(?:\s+([^>]+))?(?:\s*\/>|\s*>)/,
  (state, match, start, end) => {
    const attrs = {
      name: "FileText",
      size: 20,
    };
    
    if (match[1]) {
      const nameMatch = match[1].match(/name="([^"]*)"/); 
      if (nameMatch) attrs.name = nameMatch[1];
      
      const sizeMatch = match[1].match(/size="([^"]*)"/); 
      if (sizeMatch) attrs.size = parseInt(sizeMatch[1]) || 20;
    }
    
    const iconNode = schema.nodes.icon.create(attrs);
    tr.replaceRangeWith(start, end, iconNode);
    
    // Place selection after the icon
    tr.setSelection(Selection.near(tr.doc.resolve(start + iconNode.nodeSize)));
    
    return tr;
  }
);
```

### Fallback Behavior

If an icon name is not found:

1. A warning is logged to console
2. The default `FileText` icon is displayed
3. No error breaks the editor

**Code reference:** `components/card.tsx:60-62`

```typescript
if (IconComponent) {
  iconCache.set(normalizedName, IconComponent);
  return IconComponent;
} else {
  console.warn(`Icon "${iconName}" not found in Lucide icons.`);
  return null;
}
```
