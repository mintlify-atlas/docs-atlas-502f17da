Columns create responsive grid layouts with 1-4 columns that automatically stack on mobile devices.

## Usage

Insert columns using the `/` menu:

```
/columns
```

Or use the component tag:

```
<columns cols="2" gap="md" />
```

## Features

- **1-4 columns** - Support for up to 4 columns
- **Responsive** - Automatically stacks on smaller screens
- **Adjustable gap** - Choose from small, medium, or large gaps
- **Visual editor** - Click +/- buttons to add/remove columns
- **Dotted guides** - Visual column boundaries in edit mode

## Attributes

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `cols` | number | `2` | Number of columns (1-4) |
| `gap` | string | `"md"` | Gap size: `"sm"`, `"md"`, or `"lg"` |

## Responsive Behavior

- **1 column**: Always 1 column
- **2 columns**: 1 column on mobile, 2 on tablet+
- **3 columns**: 1 on mobile, 2 on tablet, 3 on desktop
- **4 columns**: 1 on mobile, 2 on tablet, 3 on desktop, 4 on wide screens

## Gap Sizes

- **Small (`sm`)**: 1rem (16px)
- **Medium (`md`)**: 1.5rem (24px)
- **Large (`lg`)**: 2rem (32px)

## Managing Columns

In edit mode, hover over the columns component to see controls:

1. **Decrease columns** - Click the `-` button
2. **Increase columns** - Click the `+` button
3. **Current count** - Displayed in the center

### Update Columns Function

**Code reference:** `components/columns.tsx:82-120`

```typescript
const updateColumns = useEditorEventCallback((view, newCols: number) => {
  const { getPos, node } = nodeProps;
  const pos = getPos();
  let tr = view.state.tr;
  
  const existingColumns = [];
  for (let i = 0; i < node.childCount; i++) {
    existingColumns.push(node.child(i));
  }
  
  const newColumns = [];
  for (let i = 0; i < newCols; i++) {
    if (i < existingColumns.length) {
      // Use existing column
      newColumns.push(existingColumns[i]);
    } else {
      // Create new empty column
      newColumns.push(
        view.state.schema.nodes.column.create(
          {},
          view.state.schema.nodes.paragraph.create()
        )
      );
    }
  }
  
  // Create new columns node
  const newColumnsNode = view.state.schema.nodes.columns.create(
    { ...node.attrs, cols: newCols },
    newColumns
  );
  
  tr = tr.replaceWith(pos, pos + node.nodeSize, newColumnsNode);
  view.dispatch(tr);
});
```

## Input Rule

```typescript
// columns.tsx:197-224
export const addColumnsRule = new InputRule(
  /^<columns(?:\s+([^>]+))?(?:\s*\/>|\s*>)/,
  (state, match, start, end) => {
    let cols = 2;
    let gap = "md";
    
    if (match[1]) {
      const colsMatch = match[1].match(/cols="?(\d)"?/);
      if (colsMatch) cols = parseInt(colsMatch[1], 10);
      
      const gapMatch = match[1].match(/gap="(sm|md|lg)"/); 
      if (gapMatch) gap = gapMatch[1];
    }
    
    // Create column nodes
    const columnNodes = Array.from({ length: cols }, () =>
      schema.nodes.column.create({}, schema.nodes.paragraph.create())
    );
    
    const columns = schema.nodes.columns.create({ cols, gap }, columnNodes);
    const emptyParagraph = schema.nodes.paragraph.create();
    
    const fragment = Fragment.from([columns, emptyParagraph]);
    tr.replaceWith(start, end, fragment);
    
    return tr;
  }
);
```
