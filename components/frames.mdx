Frames create bordered containers around content with customizable padding and optional captions.

## Usage

Insert a frame using the `/` menu:

```
/frame
```

Or use the component tag:

```
<frame padding="md" caption="Figure 1: Example" />
```

## Features

- **Bordered container** - Clear visual boundary
- **Adjustable padding** - Choose from 4 sizes
- **Optional caption** - Add descriptive text below
- **Click-to-edit** - Click border or caption to edit
- **Hover effects** - Visual feedback in edit mode

## Attributes

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `caption` | string | `""` | Optional caption text displayed below frame |
| `padding` | string | `"md"` | Padding size: `"sm"`, `"md"`, `"lg"`, or `"xl"` |

## Padding Sizes

- **Small (`sm`)**: 0.5rem (8px)
- **Medium (`md`)**: 1rem (16px)
- **Large (`lg`)**: 1.5rem (24px)
- **Extra Large (`xl`)**: 2rem (32px)

## Editing Frames

In edit mode:

1. **Edit settings:** Click any border edge to open the editor
2. **Edit caption:** Click the caption text
3. **Edit content:** Click inside the frame

// frame.tsx:14-39
export const frameNodeSpec: NodeSpec = {
  group: "block",
  content: "block+",
  attrs: {
    caption: { default: "" },
    padding: { default: "md" },
  },
  selectable: true,
  parseDOM: [
    {
      tag: "frame",
      getAttrs: (dom) => ({
        caption: dom.getAttribute("caption") || "",
        padding: dom.getAttribute("padding") || "md",

    },

### Padding Classes

```typescript
// frame.tsx:192-205
function getPaddingClass(padding: string) {
  switch (padding) {
    case "sm":
      return "p-2";
    case "md":
      return "p-4";
    case "lg":
      return "p-6";
    case "xl":
      return "p-8";
    default:
      return "p-4";
  }
}
```

### Input Rule

```typescript
// frame.tsx:333-353
export const addFrameRule = new InputRule(
  /^<frame(?:\s+([^>]+))?(?:\s*\/>|\s*>)/,
  (state, match, start, end) => {
    const attrs = {
      caption: "",
      padding: "md",
    };
    
    if (match[1]) {
      const captionMatch = match[1].match(/caption="([^"]*)"/); 
      if (captionMatch) attrs.caption = captionMatch[1];
      
      const paddingMatch = match[1].match(/padding="([^"]*)"/); 
      if (paddingMatch) attrs.padding = paddingMatch[1];
    }
    
    const frame = schema.nodes.frame.create(attrs, schema.nodes.paragraph.create());
    tr.replaceRangeWith(start, end, frame);
    return tr;
  }
);
```

- Hover effect in edit mode (border color changes)
- Relative positioning for clickable areas
