WebEditor provides a rich set of components for building interactive documentation and content. All components can be inserted using the `/` slash menu or by typing their input rules.

## Content Components

### Text Formatting

Inline text marks for styling content:

- **Bold** - Make text bold
- **Italic** - Make text italic
- **Strikethrough** - Strike through text
- **Code** - Format as inline code
- **Tooltip** - Add tooltips to text

[Learn more about text formatting →](/components/text-formatting)

## Inserting Components

### Using the Slash Menu

Type `/` anywhere in the editor to open the component menu. Start typing to filter components:

```
/card → Insert a card component
/code → Insert a code snippet
/callout → Insert a callout
```

### Using Input Rules

Many components support direct input rules. For example:

- Type `# ` for Heading 1
- Type `## ` for Heading 2
- Type `- ` for bullet list
- Type `> ` for quote
- Type ` ``` ` for code snippet

### Using Component Tags

You can also insert components using their tag syntax:

```
<card title="My Card" icon="FileText" />
<callout type="info" />
<accordion title="Click to expand" />
```

## Component Attributes

Most components support various attributes that can be configured:

- Click on component elements in edit mode to modify attributes
- Hover over components to see editing options
- Use attribute modals for complex configurations

See individual component pages for detailed attribute documentation.
