# WebEditor

A completely dynamic, JSX-based rich text editor powered by [ProseMirror](https://prosemirror.net/).

The alternative to BlockNote and TipTap you've been waiting for.

## What Makes WebEditor Different?

WebEditor is designed for developers who need a rich text editor that feels native to React while providing the power and flexibility of ProseMirror. Unlike other editors:

- **JSX-First**: Write custom components using familiar JSX syntax
- **Truly Dynamic**: Components are rendered as React components, not static markup
- **Markdown Native**: First-class Markdown support with MDX-like capabilities
- **Theme Aware**: Automatically adapts to your application's light/dark theme
- **Developer Friendly**: Built with TypeScript and a clean API

<Note>
  WebEditor requires React 18+ and Tailwind CSS 4+ as peer dependencies.
</Note>

## Quick Example

Get started with WebEditor in just a few lines of code:

```tsx
import { WebEditor } from "@devscribe-team/webeditor";
import "@devscribe-team/webeditor/styles";

function App() {
  return <WebEditor />;
}
```
